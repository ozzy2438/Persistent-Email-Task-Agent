# Persistent Email Task Agent

A proactive assistant that remembers incoming work, tracks open tasks, and
carries context forward across conversations.

This repository is an interview-focused applied AI project. The first milestone
sets up the Python package. The next milestones add the working agent loop,
structured output guard, two-layer memory, eval harness, API, CLI, and Docker
configuration.

## Setup

```bash
python -m venv .venv
source .venv/bin/activate
pip install -e ".[dev]"
```
